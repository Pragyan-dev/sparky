# Sparky Shopper Assets

## App Icon (icon.png)
- Size: 1024x1024px
- Design: Shopping cart with sparkle/star element
- Colors: Purple (#6750A4) and Teal (#006874)
- Style: Flat design with subtle gradients

To create the icon:
1. Use a design tool like Figma, Sketch, or Canva
2. Create a 1024x1024px square canvas
3. Design a shopping cart icon with a sparkle element
4. Use the brand colors from the theme
5. Export as PNG and save as `assets/icon.png`

## Splash Screen (splash.png)
- Size: 1284x2778px (iPhone 14 Pro Max)
- Background: Primary color (#6750A4)
- Logo: Centered app icon with "Sparky Shopper" text below
- Style: Clean and minimal

To create the splash screen:
1. Create a 1284x2778px canvas
2. Fill with primary color
3. Center the app icon
4. Add "Sparky Shopper" text below in white
5. Export as PNG and save as `assets/splash.png`

## Lottie Animations

### Loading Animation (assets/lottie/loading.json)
- Type: Spinning shopping cart or loading spinner
- Colors: Brand colors
- Duration: 2-3 seconds loop

### Success Animation (assets/lottie/success.json)
- Type: Checkmark with confetti or sparkles
- Colors: Success green (#0F9D58)
- Duration: 1-2 seconds

You can find free Lottie animations at:
- https://lottiefiles.com/
- https://iconscout.com/lottie-animations

## Product Images

Product images are referenced in `data/products.json` but not included.
For demo purposes, you can:
1. Use placeholder images from https://placeholder.com/
2. Download free food images from Unsplash
3. Create simple colored rectangles with product names
